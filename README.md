Rescue Me is a web-based roadside assistance platform that provides fast, reliable, and safe emergency services for drivers across Egypt. The platform connects users instantly with nearby tow trucks, mechanics, and auto electricians through GPS-based location tracking.

The problem addressed is the difficulty, delay, and risk drivers face when their vehicles break down, especially on highways or in unfamiliar areas. 


# Features

* Request roadside assistance services through a simple and intuitive interface
* Real-time connection between drivers and nearby service providers
* User authentication system with sign-up and login functionality
* Service request tracking and live status updates
* Driver and service provider dashboards
* Location sharing for faster assistance
* User feedback and rating system
* Responsive and modern UI/UX design
* Interactive SweetAlert2 pop-ups for a smoother user experience
